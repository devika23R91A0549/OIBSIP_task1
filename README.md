📌 Project Overview

The Iris flower dataset is one of the most popular datasets used in machine learning for classification problems. The dataset contains measurements of iris flowers belonging to three different species.

The goal of this project is to build a **machine learning model that can classify iris flowers into their correct species** based on their physical measurements.

---

## 📂 Dataset

The dataset used in this project is the **Iris Flower Dataset**, which contains measurements of iris flowers.

### Dataset Features

| Feature      | Description                           |
| ------------ | ------------------------------------- |
| Sepal Length | Length of the sepal (cm)              |
| Sepal Width  | Width of the sepal (cm)               |
| Petal Length | Length of the petal (cm)              |
| Petal Width  | Width of the petal (cm)               |
| Species      | Type of iris flower (Target variable) |

### Flower Classes

The dataset includes **three species of iris flowers**:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

---

## 🎯 Project Objectives

* Analyze the Iris dataset.
* Visualize relationships between different flower measurements.
* Train a machine learning model to classify iris flowers.
* Evaluate the performance of the classification model.

---

## 🔍 Exploratory Data Analysis (EDA)

Several data visualization techniques were used to understand the dataset and relationships between features.

Visualizations include:

* 📊 Scatter plots
* 📈 Pair plots
* 📉 Histograms
* 📦 Box plots
* 🔥 Correlation heatmap

These visualizations help identify how the different flower species vary based on their measurements.

## 🧠 Machine Learning Model

A **Support Vector Machine (SVM)** classifier was used to train the model and classify the iris flowers.

### Steps involved

1. Data loading
2. Data exploration
3. Data visualization
4. Feature selection
5. Train-test split
6. Model training
7. Prediction
8. Model evaluation

---

## 📊 Model Evaluation

The model was evaluated using **accuracy score**, which measures how many predictions were correctly classified.

The trained model successfully classifies the iris flower species based on their measurements.
